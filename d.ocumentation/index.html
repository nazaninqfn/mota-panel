نصب و راه اندازی -

بررسی اجمالی

این بخش شما را راهنمایی می کند تا تمام نرم افزارها و کتابخانه های مورد نیاز را به منظور استفاده کامل از MotaAdmin برای پروژه خود نصب کنید


نصب Node.js

Node.js® زمان اجرای جاوا اسکریپت است که بر روی موتور جاوا اسکریپت V8 کروم ساخته شده است. Node.js برای اجرای ابزارهای ساخت MotaAdmin لازم است. آخرین نسخه Node را بارگیری کرده و آن را نصب کنید: nodejs.org/en/download/


آپدیت NPM

Npm مدیر بسته JavaScript و بزرگترین رجیستری نرم افزار در جهان است. Npm یک پروژه جداگانه از Node.js است و به طور مرتب به روز می شود. در نتیجه ، حتی اگر Node.js را بارگیری کرده باشید (و بنابراین npm) ، احتمالاً باید npm خود را به روز کنید.

npm نصب --global npm@latest

همه وابستگی ها را نصب کنید

اگر می خواهید این مضمون را به روز کنید یا توسعه دهید ، باید تمام موارد وابستگی را با اجرای آن نصب کنید: 

npm نصب
